**DETH Additions** is a Mission Making Toolbox for the DETH Unit.

## FEATURES

-   Apply Loadout on Mission Start using a Preset Unit
-   Oneway Teleporter using ACE Interaction
-   Module Wrapper for BIS_fnc_spawnEnemy
